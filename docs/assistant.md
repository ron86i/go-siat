# Assistant Context

This document is specifically for the AI Assistant within the documentation system.

## Project Overview
**go-siat** is a professional SDK in Go for SIAT (Bolivia). It supports 35+ sectoral invoices and both Electronic and Computerized modalities.

## Key Features
- **Builders**: Standardized way to create complex sectoral invoices.
- **Client**: `siat.New` provides an optimized client with TraceID and Middleware support.
- **Security**: Built-in XMLDSig (digital signature) and CUF/CUFD generation.
